پردازش سریع

با معماری بهینه‌شده، محتوای شما در کسری از ثانیه پردازش می‌شود و نتایج بلادرنگ دریافت می‌کنید.

امنیت بالا

اطلاعات شما با بالاترین استانداردهای امنیتی محافظت می‌شود و هیچ‌گونه ذخیره‌سازی غیرضروری انجام نمی‌شود.

api-example.js
const analyzeContent = async (text) => {
  const response = await fetch('https://api.hooshyar.ir/v1/analyze', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': 'Bearer YOUR_API_KEY'
    },
    body: JSON.stringify({ text })
  });

  return await response.json();
};
